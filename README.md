# Multilingual Text Summarization & Translation

This project uses state-of-the-art NLP models to **summarize articles**, **generate headlines**, and **translate the results into multiple languages**. It leverages Hugging Face's transformer models and datasets to provide quick, accurate, and multilingual outputs.

---

## Features

- Summarizes long articles into concise, meaningful summaries.  
- Generates catchy and engaging hook headlines for summaries.  
- Translates summaries and hooks into multiple languages such as Hindi, French, Spanish, German, and more.  
- Demonstrates performance on real-world news datasets like CNN/DailyMail.

---

## Installation

Requires Python 3.8+ and the following libraries:

- Transformers  
- Datasets  
- PyTorch  
- NLTK  

---

## Usage

- Input an article to generate a concise summary.  
- Automatically produce a catchy headline based on the summary.  
- Translate both the summary and the headline into a target language.  

---

## Dataset

The project uses the **CNN/DailyMail** dataset for testing and demonstration, providing real-world news articles and human-written summaries for evaluation.
